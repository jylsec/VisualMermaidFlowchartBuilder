# Mermaid Flow Architect - 可视化流程图构建平台

**技术栈**: Vite | Vue 3 | Ant Design | Mermaid

## ✨ 项目概述
Mermaid Flow Architect 是一个专业级的可视化流程图构建平台，专为系统架构师、开发者和DevOps工程师设计。通过集成Mermaid的强大渲染能力和Ant Design的优雅UI，本工具让您能直观地构建复杂的系统拓扑图、微服务架构图和基础设施依赖关系图。

---

## 🎯 核心功能亮点

### 🖼️ 一体化工作区
- **沉浸式画布**：无限空间的设计画布支持复杂拓扑展示
- **悬浮画板工具栏**：常驻顶部的控制面板包含核心功能：
  - ➕ 新增六类节点
  - ⏪⏩ 撤销/重做操作
  - 💾 云端保存（持久化节点/子图数据）
  - 📸 导出高清流程图（PNG/SVG）

### 🧱 智能节点体系

### 动态关系构建
- 支持 **节点↔节点** / **节点↔域(子图)** / **域↔域** 三级连线

## 🛠️ 交互细节实现

### 🖱️ 节点操控
1. **创建节点**  
   在悬浮画板工具栏选择创建节点→ 选择节点类型 → 在画布空白区域自动创建带有默认名称即节点类型的节点

2. **单击节点** (激活悬浮节点工具栏)  
   - 删除节点及关联连线
   - 移至域：  
     选择现有子图  
     创建新域（自动命名）  
   - 移出域（保留原始连线）

3. **双击节点**  （弹出编辑对话框）
   - 自定义节点名称（e.g. "订单服务"）
   - 添加详细描述（Markdown支持）

4. **长按节点** (⏳ >500ms)  
   → 延伸虚线连接器 → 拖拽至目标 → 自动建立方向性连线

### 🧩 子图（域）管理
- **单击域** 激活悬浮子图工具栏：
  - 🗑️ 删除域（释放所有节点到根画布，保留连线）
- **双击域即子图**
  - 能够编辑域名

### 🧭 高级交互
- **框选操作**：  
  选择多个节点 → 激活批量工具栏：
  - 🗑️ 批量删除（自动清理孤立连线）
  - ➡️ 批量迁移至域（支持新建域）

- **连线操作**：  
  点击连接线 → 显示删除按钮 → 一键解除关系
